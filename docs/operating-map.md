# Operating map

| Place | Purpose | Owner action |
|---|---|---|
| GitHub issue | Mission request and acceptance criteria | Create or approve scope |
| Codex task | Atlas and specialist execution | Review Atlas reports and answer gates |
| Product branch | Isolated implementation | No routine action |
| Draft pull request | Change review and discussion | Approve, request changes, or reject |
| Company OS repository | Roles, governance, mission templates | Review operating-rule changes |

Slack, Supabase, and the Agents SDK are not required in the current phase.
